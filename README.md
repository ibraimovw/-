# 🎲 Генератор случайных предложений

Простой Python-проект, который генерирует случайные предложения, смешные фразы и забавные псевдо-факты.

---

## 🚀 Возможности

- ✍️ Генерация обычных предложений  
- 😂 Смешные предложения  
- 🧠 Забавные “факты” (сгенерированные случайно)  
- 🔁 Бесконечный цикл генерации (пока не выйдешь)  
- 📊 Статистика всех сгенерированных текстов  
- 🔒 Защита от повторения слов (не более 2 раз за запуск)

---

## 🧠 Как это работает

Программа случайным образом выбирает:
- подлежащее (subjects)
- глагол (verbs)
- объект (objects)
- обстоятельства (adverbs)
- место (places)
- причины и “факты” (reasons)

Затем собирает их в осмысленные или абсурдные конструкции.
